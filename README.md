# Harper for iTerm2

A color scheme inspired by the art of [Charley Harper][1] (1922-2007)

![Screenshot][2]

## Installation

1. Download an `.itermcolors` file of choice
2. Double-click the downloaded file
3. Open the iTerm2 preferences menu to load the preset
